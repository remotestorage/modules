Shares

Maintainer:Sebastian Kippe seba.nosp@m.stian@kip.nosp@m..pe
Version:0.3.0

Manages sharing of files.  All shares are stored with a timestamp prefix.  For images, thumbnails are created and stored in a subdirectory.
